# Repository Design

这个仓库是文章主站，不是研究原始资料库，也不是单一主题站。

## Role

- 作为 GitHub Pages 门面站。
- 发布已经形成判断的技术文章。
- 承载文章、系列和长期项目入口。
- 承载普通文章、短系列和后续 books / 专题站入口。

## Non-goals

- 不存放原始聊天、截图、日志或接口材料。
- 不替代研究资料库。
- 不把尚未收敛的调研过程包装成文章。
- 不提前把站点限定在某一个技术方向。

## Growth Path

1. 主站先承载普通文章和系列入口。
2. 当某个系列超过数篇、需要章节化导航时，拆成 books 形态。
3. 当某个主题需要数据、脚本、流程和 dashboard 时，拆成专题项目站。
